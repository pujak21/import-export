# 🌍 Import Export Trade Website

A full-stack web application for managing import/export service inquiries.

## 🚀 Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Java
- Spring Boot
- REST API

Database:
- MongoDB

## 📌 Features

- Contact / Quote Form
- Data stored in MongoDB
- REST API integration
- CORS configured
- Full-stack integration

## 🛠️ How to Run

### Backend
cd backend  
mvnw.cmd spring-boot:run  

### Frontend
Open index.html using Live Server

Backend runs on: http://localhost:8080  
Frontend runs on: http://127.0.0.1:5500
